# Security Policy

## Supported Versions

Currently, only the latest version of Net-Vanguard is supported with security updates. We recommend keeping your application up to date.

## Reporting a Vulnerability

If you've found a vulnerability, please report it via email or GitHub security advisories instead of creating a public issue. We take security very seriously and will triage your report within 48 hours.

When reporting a vulnerability, please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Possible impacts on a user's system or network.
- Any known mitigations.

You can expect an initial acknowledgment of your report within 48 hours and a timeline for a fix shortly after.

Thank you for helping keep Net-Vanguard secure!
